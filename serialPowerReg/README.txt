Regulador de tensão serial
Montado com Transistor NPN BD139 e versão simplificada do Diodo Zener 1N4735A.
Projetado para operar com entradas de 8V ~ 12V, fornece aproximadamente 5.5V.
Corrente máxima de 800mA (resistência mínima de 6.875 ohm).
Log apresenta valores de regulação de linha (~ 1.4%)

NOTAS:
Ao simular o circuito, arrume os paths de include das bibliotecas para o seu computador.
